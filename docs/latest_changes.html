(<a href=”https://redirect.github.com/pypa/cibuildwheel/issues/1372”>#1372</a>)</li> <li>🛠 Pinned version updates, including PyPy to v7.3.11 and setuptools to 66.0.0.</li> </ul> </blockquote> </details> <details> <summary>Changelog</summary> <p><em>Sourced from <a href=”https://github.com/pypa/cibuildwheel/blob/main/docs/changelog.md”>pypa/cibuildwheel’s changelog</a>.</em></p> <blockquote> <h3>v2.12.3</h3> <p><em>19 April 2023</em></p> <ul> <li>🐛 Fix an import error when running on Python 3.7.
